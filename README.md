## 🏫 学校概况

邓州市第六高级中学创建于 **1999年**，是一所位于河南省南阳市邓州市的**公办普通高级中学**。  
学校占地 **68亩**，建筑面积 **23000平方米**，环境优美，硬件设施齐全。

- **位置**：湍河国家级湿地公园河畔，西邻邓州市人民公园，东接雷锋团展览馆  
- **地址**：河南省南阳市邓州市雷锋东路（或北环路东侧）  
- **校训**：团结、奋进、求实、创新  
- **在校学生**：3800余人，42个教学班  
- **馆藏图书**：10万余册  
- **运动场地**：足球场、篮球场等一应俱全
